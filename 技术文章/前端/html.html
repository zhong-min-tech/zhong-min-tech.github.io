# canvas 数字雨

准备工作：

1、 安装VS code，已经安装的就可以跳过这一步。

2、做好立即动手敲代码的心理准备。写代码要多练多写。

基础知识：

1、html5：canvas部分

2、css

3、javascript

实现目标：

1、打字效果

2、数字雨

分析实现原理：

PART1:打字效果

打字效果的实现原理类似于连环画变成动画的原理。小时候我们在作业本画运动的小人，翻页翻得足够快，小人就会奔跑起来。

打字效果也是这样来实现的。

假如我们打印单词：wake，我们可以将步骤拆分为以下七步：

第一步：打印w

第二步：清除w

第三步：打印wa

第四步：清除wa

第五步：打印wak

第六步：清除wak

第七步：打印wake

只要将这七步执行得足够快，肉眼就会看到一个wake逐字逐字打印出来的一个效果。

PART2:数字雨

数字雨的效果需要实现以下三种变化：

文字掉落。文字掉落的实现，是通过改变文字在Y轴上的位置，实现的视觉上的掉落效果。

2、文字消失。在文字上叠加一层层透明度为0.1的黑色遮罩层就可以实现文字逐渐消失的效果。

3、文字颜色变化。通过css的hsl(),改变颜色的亮度.

全部的Js代码如下：


        //获取画布

        const canvas = document.getElementById('canvas');

        //创建绘制环境

        const ctx = canvas.getContext('2d');

        //画布宽高

        canvas.width = window.screen.width;

        canvas.height = window.screen.height;

        //背景颜色

        canvas.style.background = 'black';

        const fontsize = 26;

        ctx.fillStyle = 'green';

        ctx.font = `${fontsize}px Arial`;



        const texts = 'Wake up,Neko...:The Matrix has you...:Follow the White Rabbit:Knock,Knock,Neko...';

        const textArray = texts.split(':');

        const charArray = Array(Math.floor(canvas.width / fontsize)).fill(0);

        



        function type(text, i) {//打字效果

            if (i > text.length) {//打完退出

                return;

            }

            setTimeout(() => {

                ctx.clearRect(0, 0, canvas.width, canvas.height);

                let char = text.slice(0, i);

                ctx.fillText(char, 10, 50);

                i++;

                type(text, i);//递归调用

            }, 200)

        }



        function start() {

            type(textArray[0], 1);

            setTimeout(() => {

                type(textArray[1], 1);

            }, 3500)

            setTimeout(() => {

                type(textArray[2], 1);

            }, 8000)

            setTimeout(() => {

                type(textArray[3], 1);

            }, 16500)

            setTimeout(() => {

                digitalRain();

            }, 25000)

        }



        start();//执行代码



        function r(min, max) {//获取min-max区间内的值

            return ~~(Math.random() * (max - min + 1) + min);

        }



        function pick() {//获取数组内随机值

            return arguments[r(0, arguments.length - 1)];

        }



        function getChar() {//随机获得一个字符

            return String.fromCharCode(pick(

                r(0x109A0, 0x109FF),

                r(0x103A0, 0x103DF),

                r(0xA720, 0xA7FF)

            ))

        }



        function digitalRain() {//数字雨

            setInterval(() => {

                canvas.style.background = 'rgba(0,0,0,0.1)';

                ctx.fillStyle = 'rgba(0,0,0,0.1)';

                ctx.fillRect(0, 0, canvas.width, canvas.height);

                charArray.map((v, i) => {

                    ctx.fillStyle = `hsl(136,100%,${r(50, 78)}%)`;//改变文字亮度

                    ctx.fillText(getChar(), i * fontsize, v);

                    //改变Y轴的值

                    charArray[i] = Math.random() < 0.01 ? 0 : v + fontsize;

                })

            }, 50);

        }
